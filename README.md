# Project Overview

Malaria remains one of the world's most significant infectious diseases, requiring rapid and accurate diagnosis for effective treatment. This project investigates the application of deep learning techniques to automatically classify microscopic blood smear images as **Parasitized** or **Uninfected**.

Multiple CNN architectures were implemented and compared to identify the most effective model for automated malaria detection while emphasizing reproducibility, interpretability, and clinical relevance.
